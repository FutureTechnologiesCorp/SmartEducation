1. Для установки js библиотек/зависимсотей однажды запускаем команду при запуске приложения
npm install

2. Билдим ангуляр с отслеживанием изменений (в терминале 1)
ng build -watch

3. Билдим сборки с отслеживанием изменений (в терминале 2)
dotnet watch run

4. Для хоста проекта без .net запускаем
ng serve

Работа с usercontrols описана здесь
